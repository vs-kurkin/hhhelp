# Сервис `view`

Этот сервис предоставляет легковесный веб-интерфейс для отображения ключевой информации из `core` сервиса системы
NetAgent. Он спроектирован быть максимально простым и быстрым, предоставляя пользователю только самые важные данные о
состоянии сети.

## Технологический стек

* **Фреймворк:** [Vue.js 3](https://vuejs.org/)
* **Сборщик:** [Vite](https://vitejs.dev/)
* **Язык:** [TypeScript](https://www.typescriptlang.org/)
* **Веб-сервер (в Docker):** [Nginx](https://www.nginx.com/)

## Основные команды

Для управления сервисом используются npm-скрипты, определенные в `package.json`:

* **Локальная разработка:**
  ```bash
  npm run dev
  ```
  Запускает сервер для разработки с горячей перезагрузкой.

* **Сборка для production:**
  ```bash
  npm run build
  ```
  Собирает и минифицирует исходный код в директорию `dist` для развертывания.

* **Локальный просмотр сборки:**
  ```bash
  npm run preview
  ```
  Запускает локальный сервер для просмотра production-сборки из папки `dist`.

## Развертывание

Сервис контейнеризован с помощью Docker. `Dockerfile` состоит из двух этапов:

1. **`builder`**: Устанавливает зависимости и собирает проект с помощью `npm run build`.
2. **Финальный образ**: Копирует статичные файлы из `dist` в легковесный образ `nginx:stable-alpine`.

Для запуска в составе всего проекта используется `docker-compose.yml`. Сервис доступен по порту, который пробрасывается
на 80 порт контейнера.
